\subsection{Модуль statistics}

\url{https://docs.python.org/3/library/statistics.html}

1) Средние значения

\begin{minted}{python}
import statistics as st
x = [1, 2, 3, 4, 4]
st.mean(x)    # среднее арифм. (average)
st.median(x)  # медиана (middle value)
st.mode(x)    # мода (самое частое значение)
\end{minted}

Медиана:
\begin{minted}{python}
>>> st.median([1, 3, 100])
3
>>> st.median([1, 3, 5, 100])
4.0  # интерполируется, если кол-во элементов четно
>>> st.median_low([1, 3, 5, 100])
3
\end{minted}


2) Разброс
\begin{minted}{python}
st.pvariance()  # дисперсия D
st.pstdev()  	 # standard deviation (СКО) sigma
st.variance(x)  # дисперсия выборки: D0 = D * sqrt(N/(N-1))
st.stdev(x)  	 # standard deviation (СКО) выборки
\end{minted}

$\sigma(X) = \sqrt{D(X)}$
