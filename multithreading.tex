\section{Многопоточность}$~$

Два вида многозадачности: 
\begin{itemize}
 \item на основе процессов (организуется для параллельного выполнения программ);
 \item на основе потоков (для параллельного выполнения отдельных частей одной программ).
\end{itemize}

Многозадачность \emph{на основе процессов} --- средство, благодаря которому на компьютере могут параллельно выполняться две программы и более.

\emph{Поток} --- часть исполняемого кода программы.


1) Реализация в C\#:

\verb|System.Threading|  --- пространство имен, которое содержит различные типы, позволяющие создавать многопоточные приложения.

2) Реализация в Python:

Многозадачность:
\begin{itemize}
 \item ветвление процессов \verb|fork/exec|
\end{itemize}

Многопоточность:
\begin{itemize}
 \item низкоуровневый модуль \verb|_thread|
 \item высокоуровневый модуль \verb|threading|
\end{itemize}
