\subsection{split and join}$~$

\verb|str.split(delimiter?, maxsplits?)| --- разбитие строк по пробельный символам (пробел, табуляция, перевод строки) либо с указанием разделителя

\verb|delimiter.join(iterable)| --- объединяет последовательность подстрок, добавляя разделитель delimiter.

\begin{lstlisting}
>>> t = 'Hello    World   !'
>>> t.split()
['Hello', 'World', '!']
>>> s = t.split()
>>> '---'.join(s)
'Hello---World---!'
\end{lstlisting}


\subsection{Шаблоны. Модуль string}$~$

\begin{lstlisting}
>>> import string
>>> vals = {'F': 'Ivanov', 'N':'Ivan'}
>>> t = string.Template('---$F---$N----')
>>> t.substitute(vals)
'---Ivanov---Ivan----'
\end{lstlisting}


\subsection{Модуль re}$~$

\verb|import re|

Основные функции модуля re:
\begin{itemize}
 \item \verb|re.compile(pattern, [flags,] string)| --- собирает регулярное выражение в отдельный объект;
 \item \verb|re.match(pattern, string)| --- ищет по заданному шаблону в начале строки;
 \item \verb|re.search()|
 \item \verb|re.findall()|
 \item \verb|re.split()|
 \item \verb|re.sub()|
\end{itemize}

Варианты использования

1) без комплирирования:

\begin{lstlisting}
text = "Hello 1213213 132 world"
mobj = re.match('Hello (.*) world', text)
print(mobj.group(0)) # Hello 1213213 132 world
print(mobj.group(1)) # 1213213 132
\end{lstlisting}

2) с комплирированием шаблона (работает быстрее):

\begin{lstlisting}
text = "Hello 1213213 132 world"
patt_obj = re.compile('Hello (.*) world')
mobj = patt_obj.match(text)
print(mobj.group(0)) # Hello 1213213 132 world
print(mobj.group(1)) # 1213213 132
\end{lstlisting}
